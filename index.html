一、首发功能：两大核心 Skills

HTX AI Skills 首发版本包含两个核心 Skill，覆盖从交易指令到执行的完整流程。

  • 现货交易：支持执行现货交易操作，包括市价单、限价单、下单与撤单、查询订单状态。
  • 合约交易：支持执行合约交易操作，包括开多、开空、平仓、调整杠杆以及设置止盈止损。

 

二、HTX AI Skills 核心亮点

  • 自然语言交易，零门槛上手：无需学习复杂 API 或编写代码，只需用自然语言描述交易意图，AI 即可理解并执行。
  • 现货与合约双场景支持：首发版本覆盖现货与合约两大核心交易场景，满足不同交易需求。
  • 多工具兼容，即插即用：支持 OpenClaw、Claude Code、Cursor 等主流 AI Agent 开发工具，3 分钟即可开始交易。
  • 开放协议，持续扩展：HTX AI Skills 采用标准 Skill 协议架构，未来将持续扩展更多能力模块，不断提升 AI 调用体验。
  • 面向所有用户：无论是普通用户、AI 开发者还是量化交易者，HTX AI Skills 都可以成为专属 AI 交易助手。

 

三、使用教程

仅需三步即可快速上手 HTX AI Skills。

第一步：打开 AI 开发工具

支持以下主流工具：

  • OpenClaw
  • Claude Code / Claude CLI
  • Codex / Codex CLI
  • Cursor
  • 以及更多兼容工具

第二步：一行命令接入 Skills

在终端输入以下命令即可接入：

npx skills add https://github.com/htx-exchange/htx-skills-hub

第三步：使用自然语言调用 Skill

  • 现货交易示例：“用 1000 USDT 买入 BTC”、“以 150 美元挂单买入 SOL”、“卖出我所有的 ETH”
  • 合约交易示例：“开多 BTC，10 倍杠杆”、“做空 ETH，仓位 500 USDT”、“平掉我的 SOL 空单”

使用交易功能需配置 HTX API Key。

 

四、未来规划

  • MCP Server：火币HTX将推出官方 MCP Server，支持 Claude、Cursor、OpenClaw 等主流 AI 客户端原生接入。相比 Skills 的文档模式，MCP Server 将显著提升 AI 调用稳定性和执行成功率，使 AI Agent 具备生产级交易能力。
  • CLI 命令行工具：面向开发者与量化交易者推出 CLI 命令行工具，支持脚本调用、自动化工作流以及定时任务，并提供标准化 JSON 输出，可无缝接入策略系统和数据管道。
  • 更多 Skills 模块：未来 HTX AI Skills 将持续扩展更多能力，包括账户管理、交易历史、行情指标、跟单交易、理财产品、活动查询等功能。
  • 彩蛋：HTX App AI 助手即将上线，支持通过自然语言查询行情、解读市场趋势，并获取交易与理财相关信息。
